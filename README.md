# aklry-cli

![npm 版本](https://img.shields.io/badge/yuan-cli_v0.4.1-green)

## 简介

- 用于快速构建前端项目的命令行工具

## 模板

- 快速生成各种项目模板,包含(后台管理系统、uniapp项目、h5项目)

## 安装

```bash
npm install -g aklry-cli
```

## 使用

```bash
# 使用命令创建模板(新项目)
yuan create
yuan create [projectName]
```

```bash
# 使用命令打包部署
yuan run
```

```bash
# 使用命令创建组件
yuan add-cpn Navbar src/components
```

## 查看脚手架版本

```bash
yuan -v
yuan --version
```
